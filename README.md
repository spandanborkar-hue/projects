# 🍔 Food Ordering Website
A fully responsive, single-page food delivery application. Users can browse categories, manage a real-time shopping cart, and simulate a checkout process.

https://react-spandan-food-ordering.netlify.app

## ✨ Key Features
* **Dynamic Menu:** Food items loaded dynamically with category filtering (e.g., Pizza, Burgers).
* **Real-time Cart:** Add, remove, and adjust quantities with automatic subtotal/tax calculation.
* **Search Functionality:** Filter through dishes instantly.
* **Responsive UI:** Optimized for mobile, tablet, and desktop using Tailwind CSS.
* **User Feedback:** Form validation for checkout.

## 🛠️ Tech Stack
* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS v4
* **State Management:** Redux, Context API, standard Hooks.
